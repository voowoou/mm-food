# MM FOOD

This app is under development...

## Features

### User experience

In the future users will be able to:

-   search recipes using filters (cuisines, untolerances, calories, etc.);
-   find certain meals using search bar;
-   check author's menu paired with music albums.

### Frontend approaches

Whean the basic code is finished, different approaches can be applied to coding of the app pages.<br/>
For instance:

-   The page with recipes filters is going to have a server side rendering (SSR);
-   The page with meal searching is going to have a client side rendering (CSR);
-   The page with meal and music pairing is going to have a static site generetion (SSG).<br />
Approaches can be changed. They depend on final features of the page and hosting.
