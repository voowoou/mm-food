# MM FOOD

This app is under developing...

## Features

### User experience

In the future users will be able to:

-   search recipes using filters (cuisines, untolerances, calories, etc.);
-   find a certain meal from search bar;
-   check author's menu paired with some music albums.

### Frontend approaches

As soon as the basic code will be ready there is an idea to apply different approaches to coding of the app pages.<br/>
For instance:

-   The page with recipes filters is going to have a server side rendering (SSR);
-   The page with the meal searching is going to have a client side rendering (CSR);
-   The page with the meal and music pairing is going to have a static site generetion (SSG).
    Approaches can be changed. It depends on final features of the page and a hosting.
